Manjaro

Fix Manjaro's Keyring

When keyring is corrupted, you can fix it by running the following commands:

sudo pacman -Syy sudo pacman-key --refresh-keys sudo pacman-key --populate archlinux manjaro sudo pacman -Sc

How to Import Certificates

To import certificates, you need to store them in the ~/.ssh/certs directory and run the following commands:

sudo trust anchor --store ~/.ssh/certs/<cert-name>.crt sudo update-ca-trust

Check your imported certificates:

awk -v cmd='openssl x509 -noout -subject' '/BEGIN/{close(cmd)};{print | cmd}' < /etc/ssl/certs/ca-certificates.crt

Fix allows non-root process to bind to privileged ports (443,80)

Run as root the following command:

sudo su setcap CAP_NET_BIND_SERVICE=+eip /usr/bin/ssh

Bottles Flatpak

Can't enable Steam Proton manager

Starting from version 2022.6.28, Bottles allows you to manage Steam Proton prefixes by enabling the setting of the same name. If you've installed Bottles via Flatpak, it is essential to give it permissions to access the Steam path; otherwise the integration will not be activated. Proceed, according to your Steam installation.

flatpak override --user com.usebottles.bottles --filesystem=xdg-data/Steam

Pipewire

Fix auto-switching

rm -rf ~/.config/pulse

Fix audio stuttering (account-wide)

mkdir -p ~/.config/wireplumber && cp /usr/share/wireplumber/*.conf ~/.config/wireplumber mkdir -p ~/.config/pipewire && cp /usr/share/pipewire/*.conf ~/.config/pipewire

Fix audio stuttering (system-wide)

sudo mkdir -p /etc/wireplumber && sudo cp /usr/share/wireplumber/*.conf /etc/wireplumber sudo mkdir -p /etc/pipewire && sudo cp /usr/share/pipewire/*.conf /etc/pipewire

Get allowed rates in your system

grep -E 'Codec|Audio Output|rates' /proc/asound/card*/codec#*

Firefox

In 'about:config' set the following variables:

Fix Video Playback

general.useragent.override

Select 'String' and click '+' and add:

Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:106.0) Gecko/20100101 Firefox/106.0

Fix CSV Password File Loading

Set variable 'TRUE' to enable csv file importing.

signon.management.page.fileImport.enabled

Set variable to 'TRUE' to force Firefox to read the csv file.

security.allow_eval_in_parent_process

SonarQube

Get all limits:

ulimit -a

Fix

sudo vim /etc/sysctl.d/99-sysctl.conf

Add the following lines:

vm.swappiness=10 vm.max_map_count=262144 fs.file-max=131072

Docker

Fix library initialization failed, unable to allocate file descriptor table, out of memory error

sudo vim /usr/lib/systemd/system/docker.service

Choose one value (65536:65536 or 131072:131072) setting and add it to the file:

[Service] ExecStart= ExecStart=/usr/bin/dockerd --default-ulimit nofile=<value>:<value> -H fd://

Save and run:

sudo systemctl daemon-reload sudo systemctl restart docker

Oracle Database

How to run Oracle Database on Manjaro

Find a user's UID:

id -u <username>

Find a user's GID:

id -g <username>

List all groups a user belongs to:

id -G <username>

If you wish to see the UID and all groups associated with a user, enter id without any options, as follows:

id <username>

Create a group with specific ID

sudo groupadd -g 54321 oinstall

Create a user account with specific ID and groups, without a Home directory

sudo useradd -Mu 54321 -G oinstall,docker,dev,matias,admin,coder oracle

Then lock the account to prevent logging in

sudo usermod -L oracle
